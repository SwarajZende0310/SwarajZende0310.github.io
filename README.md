# Swaraj Zende — Personal Website

A playful, responsive portfolio for a robotics engineer. Built with plain HTML,
CSS, and JavaScript so it can be served directly by GitHub Pages without a
build step.

## Run locally

Open `index.html` directly, or serve the directory:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Customize

- Page content and project links live in `index.html`.
- Colors and visual styling live in `style.css`.
- Interactions—including the randomly placed sleepy mascot—live in `script.js`.
